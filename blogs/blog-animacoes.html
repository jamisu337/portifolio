Animações com CSS & GSAP

As animações não são apenas "enfeites" em uma interface. Elas servem como feedback visual, guiando o usuário e tornando a experiência mais orgânica. No meu workflow, utilizo uma combinação de CSS puro para transições simples e GSAP para interações complexas.

O Poder dos Keyframes

O exemplo acima foi construído utilizando apenas CSS. O segredo está na propriedade @keyframes, que permite definir estados específicos para um elemento ao longo do tempo.

@keyframes pulse-rotate {
  0% { transform: scale(1) rotate(0deg); }
  50% { transform: scale(1.2) rotate(180deg); }
  100% { transform: scale(1) rotate(360deg); }
}

Para animações que dependem do scroll do usuário ou sequências encadeadas, o GSAP (GreenSock Animation Platform) torna-se indispensável, permitindo um controle de linha do tempo que o CSS padrão ainda não alcança com facilidade.

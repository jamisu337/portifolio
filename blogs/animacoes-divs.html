5 Animações Essenciais com CSS

Uma coleção de snippets prontos para copiar e colar.

1. Rotate (Giro Infinito)

.box {
  animation: rotate 3s infinite linear;
}

@keyframes rotate {
  0% { transform: rotate(0deg); }
  100% { transform: rotate(360deg); }
}

Ideal para ícones de carregamento (loading) ou elementos decorativos em background.

2. Pulse (Batimento)

.box {
  animation: pulse 2s infinite;
}

@keyframes pulse {
  0%, 100% { transform: scale(1); }
  50% { transform: scale(1.4); }
}

Ótimo para botões de CTA (Call to Action) para chamar a atenção do usuário.

3. Slide (Deslizamento)

.box {
  animation: slideIn 2s infinite;
}

@keyframes slideIn {
  0% { transform: translateX(-50px); }
  50% { transform: translateX(0); }
  100% { transform: translateX(50px); }
}

Usado frequentemente em carrosséis ou para revelar conteúdo ao fazer scroll.

4. Fade In/Out (Opacidade)

.box {
  animation: fadeIn 2s infinite;
}

@keyframes fadeIn {
  0%, 100% { opacity: 0.2; }
  50% { opacity: 1; }
}

A transição mais elegante. Essencial para modais e tooltips.

5. Bounce (Pulo)

.box {
  animation: bounce 2s infinite;
}

@keyframes bounce {
  0%, 100% { translateY(0); }
  40% { translateY(-30px); }
  60% { translateY(-15px); }
}

Muito comum em ícones de notificação ou setas indicando "role para baixo".
